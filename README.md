## 使用说明
1. 安装编译
```shell script
# 拉取代码
git clone https://github.com/dnsjia/luban.git

# 打包
cd luban
go build main.go -o ./luban
or
GOOS=windows GOARCH=amd64 go build main.go ./luban

# 启动
./luban

# 启动前端
cd luban/luban_fe
npm run dev
```

2. 启动服务前先创建etc/config.yaml
```shell script
# 数据库配置
mysql:
  path: '192.168.1.96:3306'
  db-name: 'luban'
  username: 'root'
  password: '123456'
```

3. 初始化数据库
```go
windows执行以下脚本, 初始化数据库

init_db.bat
```
